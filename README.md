# RSS Reader

RSS Reader - это приложение для чтения RSS-потоков, построенное на промисах.

## Демо

Посетите [RSS Reader на Vercel](https://rss-reader.vercel.app)

## Особенности

- Добавление RSS потоков по URL
- Валидация URL
- Проверка на дубликаты
- Отображение последних постов из RSS
- Адаптивный дизайн с Bootstrap

## Установка

```bash
git clone https://github.com/yourusername/rss-reader.git
cd rss-reader
npm install
